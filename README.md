# projeto-de-software-ConectaAlimento
Projeto acadêmico da disciplina de Projeto de Software (Engenharia de Software - PUC Minas). O repositório contém a modelagem UML para um Sistema de Gerenciamento de Doação de Alimentos (ConectaAlimento), focado em gerenciar doações de estabelecimentos, coletas por voluntários e a distribuição para ONGs.
